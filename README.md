# PySlide

Presentation Website with interactive Python code cells (similar to Jupyter Notebook).

---

> [!NOTE]
> This website was completely created via AI. There is no support.

## Features

- **Interactive Canvas**: Drag and drop elements to create slides.
- **Python Integration**: Run Python code directly in your slides using Pyodide.
- **Rich Text Editing**: Powerful text tools with custom fonts and styles.
- **GitHub Sync**: Cloud synchronization using your own GitHub token.
- **Presentation Mode**: Full-screen mode with responsive scaling for any display.

## How to use

1.  Open the application.
2.  Create your slides.
3.  Add code cells and run them.
4.  Sync with GitHub to save your progress across devices.

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- [Pyodide](https://pyodide.org/) (Python in the browser)
- [CodeMirror](https://codemirror.net/) (Code editor)
- GitHub REST API for synchronization
